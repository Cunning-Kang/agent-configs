## Agent skills

Maintenance documentation for the agent skills (issue tracker, triage labels, domain conventions) lives under `docs/maintenance/`. See `CONTEXT.md` for the architecture overview.

### Issue tracker

Issues tracked in GitHub via `gh` CLI. See `docs/maintenance/issue-tracker.md`.

### Triage labels

Five-label vocabulary: needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix. See `docs/maintenance/triage-labels.md`.

### Domain docs

Single-context layout. See `docs/maintenance/domain.md`.
